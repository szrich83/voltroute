VoltRoute

VoltRoute is a web-based EV trip planning and charging estimation tool built with F# and WebSharper.

The application helps electric vehicle users estimate:

trip energy consumption

required charging

charging time

trip electricity cost

Technology stack

F#

WebSharper

ASP.NET

Vite

.NET 10

Course context

This project is developed as part of the course:

Introduction to Functional Programming in F#
University of Dunaújváros

Project Alpha goal

Create a functional EV trip planner implemented as a web application using F#.

Running locally

Requirements:

.NET 10 SDK

Node.js
